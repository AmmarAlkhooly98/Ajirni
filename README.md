### Installing Dependencies

From within the root directory:

```bash
pip install pipenv
pipenv install
cd frontend 
npm install 
```
### Running the server

```bash
pipenv shell
cd yama
cd ajirni
python manage.py runserver
```

### Running the Front-End server

```bash
cd frontend
npm start
```
